## 1. 入門必修：ブラックボックスを解体する「ノード思考」

初めて ComfyUI を開くと、画面いっぱいのクモの巣のようなワイヤーに圧倒されるかもしれません。しかし、「ノード (Nodes)」のロジックさえ理解すれば、これが世界で最も自由な積み木であることに気づくはずです。

- **Node (演算ノード)**: 感情のない作業員です。モデルをロードする人、プロンプトを解釈する人、データを画像としてレンダリングする人がいます。
- **Link (データケーブル)**: ノード間の線は「純粋なエネルギーと情報」を伝達します。「モデルノード」の出力を「レンダリングノード」の入力に差し込んだ瞬間、火花が散り、創造が始まります。
- **Workflow (ワークフロー)**: これは私たちの「自動化台本」です。すべての接続構成は最終的に `.json` ファイルとして保存され、いつでも神レベルの描画スクリプトをロードできます。

---

## 2. 戦闘準備：システムを汚さない「ポータブル環境」

Windows で重い AI ソフトを動かす際、最も避けたいのはシステム環境を汚すことです。私の助言を聞いてください。**「Portable（ポータブル版）」一択です！**

```text
ComfyUI_windows_portable/
├── ComfyUI/
│   ├── models/             # チェックポイント（モデル）はここ
│   ├── custom_nodes/       # 各種プラグインはここ
│   └── output/             # 生成された画像はここ
├── python_embeded/         # 独立した Python 環境（システムを汚さない）
└── run_nvidia_gpu.bat      # 起動ボタン
```

> 💡 **VRAM が少ない場合 (< 6GB)**:
> 無理は禁物です。起動時に `--lowvram` パラメータを付けてください。エンジンに冷却器を付けるようなもので、少し遅くはなりますが、グラボが悲鳴を上げて止まることは防げます。

## 3. 必須装備：ComfyUI Manager (管理者の眼)

手動で依存関係を解決する苦行から解放されたいなら、真っ先に **ComfyUI Manager** をインストールしましょう。

ネットで拾ったクールなワークフロー JSON を開いたとき、画面が「赤いエラーノード」だらけになっても、**パニックにならないでください。** Manager の **`Install Missing Custom Nodes`** をクリックすれば、不足しているプラグインを全自動でスキャンし、インストールしてくれます。これこそがエレガントな開発スタイルです。

---

## 4. プログラマーから経営者へ：エンジンを API の奴隷にする

ここが今回の最も価値あるポイントです。Moyin のバックエンドから ComfyUI に自動で絵を描かせる方法を解説します。

### Step 1: 開発者モードの開放

Settings から **「Enable Dev mode Options」** をオンにします。すると、**「Save (API Format)」** というボタンが現れます。ここで保存される JSON こそが、UI の座標情報を削ぎ落とした、純粋な「プログラム用台本」です。

### Step 2: Python による遠隔操作

数行のスクリプトで、ComfyUI を「見えない形」で制御できます。

```python
import json, urllib.request

# API 形式のワークフローをロード
with open("beauty_workflow_api.json") as f:
    wf = json.load(f)

# プロンプトを動的に書き換える
wf["6"]["inputs"]["text"] = "A cool cyber-cat, neon city, 8k"

# サーバーへ送信
req = urllib.request.Request("http://127.0.0.1:8188/prompt",
                             data=json.dumps({"prompt": wf}).encode())
urllib.request.urlopen(req)
```

NPC の服を着せ替えたい？バックエンドでキーボードを叩けば、AI が自動で描画し、ゲーム画面へ送り返してくれます。これこそが真の AI 活用です。

> 🗣️ `「AI 助教！完璧な ComfyUI API ワークフロー JSON は用意した。
今すぐ Python の管理スクリプトを書け。第一に、VRAM が溢れないよう【キュー（Queue）管理機能】を実装しろ。
第二に、【ノードパラメータの動的オーバーライド】を導入しろ。呼び出し時に LoRA の強度やキーワードを自由に変えたい。

さあ、全自動画像生産ラインのコードを吐き出せ！」`

このフローをマスターすれば、あなたは単なる「絵描き」を卒業し、無限の計算資源を持つコンテンツ工場の「オーナー」へと進化します。
