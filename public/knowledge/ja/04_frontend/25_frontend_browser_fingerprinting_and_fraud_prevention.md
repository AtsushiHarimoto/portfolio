# 透明の牢獄：ブラウザ・フィンガープリントと無感防詐欺 (Browser Fingerprinting)

## @Overview

こんにちは、AKIRA です。
今回は業界の暗黒魔術について話しましょう：**不正防止技術** です。

シナリオを想像してください：あなたの API は CORS 保護があり、あなたのサイトからしかアクセスできないようになっています。しかしハッカーが「Chrome ブラウザを完璧に偽装した Python スクリプトを自分で書いて」データを猛烈にスクレイピングし、1秒間に 1万件のゾンビ会員登録までしてきたら、どうしますか？

今回は、使い捨てアカウントを完全に暴く **ブラウザ・フィンガープリント (Fingerprinting)** と、現代サイトの究極の門番 **無感知 reCAPTCHA / デバイス証明** を解き明かします。

## 前書き：Cookie は消せても、あなたの「指紋」は永遠に消えない

通常、私たちは JWT、Cookie、または LocalStorage を使って「过去に私たちのサイトを訪れた人が誰なのか」を記憶します。しかしこれらは、ハッカーが「シークレットモード（プライベートブラウズ）」を使ったり、削除ボタンを押すだけで瞬殺されます。結果として同じ転売屋が自動化スクリプトでループ処理して Cookie をクリアし、コンサートのチケット販売サイトで 1万個の抽選アカウントを再登録することが可能になります。

しかし彼らが知らない残酷な真実があります：**彼らのハードウェアは、ページが読み込まれた 0.1秒の間に、すでに「絶対的な身分証明」をあなたのフロントエンドコードに差し出しているのです。この技術を、業界では「ブラウザ・フィンガープリント」と呼びます。**

## 1. 逃げ場のないピクセルと波長：指紋はどう計算されるか？

フロントエンドエンジニアにはハードディスクを読む権限はありませんが、一見無害に見えるブラウザ API に数十種類アクセスできます。この数十の微細な特徴点を組み合わせると（これを Entropy：情報エントロピーと呼びます）、この星上に全く同じ二人の人間は事実上存在しなくなります！

### 🎨 最も強力な追跡法：Canvas 画布フィンガープリント

これが現在最も悪名高く、最も防御しにくい追跡手法です。

- JavaScript が画面外でこっそり「グラデーション、特殊フォント、各国の絵文字がぎっしり詰まった」非表示の 2D Canvas 要素を描画します。
- 天才的な部分がここ：**世界中の人間が異なる GPU ブランド（NVIDIA か AMD か）、ドライバーバージョン、OS の文字描画エンジン（Windows か Mac か）を持っているため、この画像上の同じピクセルエッジの「アンチエイリアス数学演算」に、顕微鏡レベルの差異が生まれます！**
- 描画された画像をハッシュ値に変換します。ハッカーがPC を換えない限り、シークレットモードを使おうと、どんなブラウザに変えようと、VPN で IP を変えようと、彼の Canvas Hash 値は ほぼ 100% 全く同じになります。彼に逃げ場はありません。

### 🎵 音声とハードウェア指紋の連合攻撃

描画だけでなく、JS は密かにコンピュータのサウンドカードに、10ミリ秒の聞こえない高周波音を生成させることができます。それを再生するのではなく、逆にそのサウンドチップがこの音を処理した「微分数学的特徴」（AudioContext フィンガープリント）を分析します。加えて CPU コア数（`navigator.hardwareConcurrency`）やサブディスプレイの解像度なども覗き見ます。

これら数十のハードウェア特徴をすべて組み合わせると：**おめでとうございます、あなたのネット上の身体は完全に捕捉・追跡されています。**

## 2. ボットを門の外で締め出す：現代の究極不正防止エンジン

あなたの Moyin ログインシステムがボットに毎秒 10万回のクレデンシャルスタッフィング攻撃を受けたとき、IP ブロックだけでは到底解決できません（ハッカーが持っているゾンビ IP はあなたの知人より多いです）。

### 🤖 伝統的で苦痛な画像 CAPTCHA

以前は怒りを覚える「全ての信号機が含まれた画像を選んでください」タイプのものを表示していました。これは本物のユーザー体験の大きな破壊です。さらに悲惨なことに、現代の AI ビジョンモデルは横断歩道や消火栓を人間より正確に認識します。信号機の九宮格は正式に死亡宣告されました。

### 👻 無感防御の救済：reCAPTCHA v3 / Cloudflare Turnstile

これこそが現代アーキテクチャの救済の道！ページ上に「私はロボットではありません」のチェックボックスすら存在しません。**では、どうしてあなたが人間だとわかるのでしょう？**

1.  **マウス軌跡の数学**: 本物のマウス移動には、パーキンソン病のような微小な人間の手の震えとベジェ曲線が含まれます。Selenium で書かれたクローラーのマウス移動は完璧に直線的か、または機械的なアルゴリズムのランダム性しかありません。
2.  **クロスドメイン指紋の交差**: Google のスクリプトが密かに確認します：このタブを開く前に正常なネット閲覧履歴がありますか？ この「GPU フィンガープリント」で 30分間 YouTube を見続けた実際のアカウントですか？
3.  最も強烈なのが、Cloudflare Turnstile が Web Workers を使ってブラウザに「CPU 集中型の数学パズル（Proof-of-Work）」を解かせます。この程度の計算量は一般人のスマートフォンを落とすことはありませんが、格安クラウドホストを借りて 1,000個のクローラーウィンドウを一度に開こうとするハッカーのCPUを即座に焼き切り破産させます。

これら 3つの技術がスコアリングすると、フロントエンドに「信頼性証明トークン」が生成されます。あなたのバックエンド API はこの **合格した信頼性トークンを持つ登録リクエストのみ** を受け付けます。
