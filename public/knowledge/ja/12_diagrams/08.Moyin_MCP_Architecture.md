# MCP サーバー・システムアーキテクチャ (MCP Architecture)

## @Overview

`moyin-mcp-server` は、フロントエンド UI とバックエンドの各種レンダリングサービス（TTS, ComfyUI, Video Render）を統合する、プロジェクトの核心的な調度中枢です。

## 📡 通信プロトコルとデータ制御

1.  **Command Channel (コマンドチャンネル)**:
    生産指示（例: `start_render`）を送信するためのエンドポイント。
2.  **Status Sync (ステータス同期)**:
    Redis Pub/Sub に基づくリアルタイムなステータス同期。フロントエンド UI はタスクの進捗状況をリアルタイムでリスニングします。
3.  **Data Sync (データ同期)**:
    各ノード間のデータ交換フォーマットとして、独自定義の `StoryPack` を採用し、データの整合性を担保します。
