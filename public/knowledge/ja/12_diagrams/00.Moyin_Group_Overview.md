## 1. 幻海師範の道場：Moyin Model Studio

**例え話**: 霊光波動拳の修練場
**場所**: `workspace/knowledge/00_projects/moyin-model-studio`
**ビジュアルマップ**: [01.Reikai_Dojo.canvas](../00_projects/moyin-model-studio/01.Reikai_Dojo.canvas)

ここは幻海師範の道場のような場所で、過酷な試練に満ちています。普通の霊気（データ）ではここでは持ちません！
**基礎霊力訓練（Pipeline）** を経て不純物を取り除き、純粋な霊気を精製する必要があります。
そして、歴代の強者が残した**霊光玉（Base Models）** を継承し、最終的な**奥義伝承（Training）** を行います。
**試練の門（Evaluation）** を突破した強者だけが「霊光波動拳」をマスターし、一人前の戦力となれるのです！

- **Raw Data** -> **霊力資質**
- **Pipeline** -> **基礎訓練**
- **LoRA/Model** -> **霊光波動拳の奥義**

## 2. 魔界整形師の手術室：Moyin ComfyUI

**例え話**: 時雨のクラブ
**場所**: `workspace/knowledge/00_projects/moyin-comfyUI`
**ビジュアルマップ**: [01.Makai_Surgery.canvas](../00_projects/moyin-comfyUI/01.Makai_Surgery.canvas)

うわぁ、ここは不気味な雰囲気ですね……ここは魔界整形師・時雨の縄張りです。
参加者が外見を変えたり、特殊な能力（画像生成）を得たい場合は、ここで改造を受けなければなりません。
強烈な**執念（Prompt）** を代償として提出すると、時雨が精密な**手術術式（Workflow）** を設計します。
さまざまな**妖力増幅器（Custom Nodes）** が追加されることもあり、最後に……ドン！**具現化（Generation）** 完了！

- **Prompt** -> **執念 / 代償**
- **Workflow** -> **手術術式**
- **Image** -> **改造後の肉体**

## 3. 妖狐蔵馬の魔界植物園：Index-TTS

**例え話**: 魔界植物の召喚
**場所**: `workspace/knowledge/00_projects/index-tts`
**ビジュアルマップ**: [01.Kurama_Garden.canvas](../00_projects/index-tts/01.Kurama_Garden.canvas)

しっ……静かにしてください。ここは蔵馬の秘密の花園です。
一見美しい植物に見えますが、実は魔界の生物なのです！
蔵馬は**前世の記憶（Reference Audio）** を触媒として使い、強大な**妖力（Inference）** を注入します。
あらゆる声を模倣できる**魔界オジギソウ（Voice Cloning）** を生み出すのです。聞こえますか？あの魅惑的で危険な声が……

- **Reference Audio** -> **召喚触媒**
- **Inference** -> **妖力注入**
- **TTS Output** -> **魔界植物の声**
