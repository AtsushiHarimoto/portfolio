# ターミナルの「黒魔術」：PowerShell ショートカット別名ガイド (Alias Cheat Sheet)

## @Overview

エンジニアの生産性を決定づけるのは、タイピングの速度ではなく「不要なタイピングをいかに減らすか」にあります。Moyind のような複数のマイクロサービスが絡み合うプロジェクトでは、ディレクトリの移動やサービスの起動といった高頻度なアクションを「別名 (Alias)」として登録し、一瞬で実行できる環境を整えることが基本戦略となります。本稿では、開発効率を劇的に向上させるための基本設定と、魔法のようなショートカットコマンドについて解説します。

---

## 1. 冗長な作業の排除：Alias 設定の重要性

複雑なパスを記憶したり、長い起動コマンドを毎回手入力したりするのは、ヒューマンエラーの温床であり、認知リソースの浪費です。

### 🛠️ 設定の仕組み

Windows では `$PROFILE`（PowerShell の設定ファイル）、macOS/Linux では `.zshrc` や `.bashrc` に設定を書き込むことで、特定の文字列をコマンド群に紐付けることができます。

- **メリット**:
  1.  **ミス防止**: 正確なパスや引数を事前に定義しておくことで、起動失敗をゼロにします。
  2.  **即時点火**: 開発の開始時に 2〜3 文字を打ち込むだけで、すべてのサービスが立ち上がる爽快感を実現します。

---

## 2. 推奨ショートカットコマンド清單 (Core Command List)

Moyind プロジェクトにおいて、毎日実行することになる「主要三系統」の魔法を紹介します。

### 🚀 ① 瞬間点火とディレクトリ移動 (Startup & Navigation)

- **`mo-api`**: `projects/moyin-gateway` ディレクトリへ瞬時に移動し、ランタイム環境の初期化と API 起動の準備を整えます。
- **`mo-web`**: フロントエンド管理画面である `projects/moyin-web` の戦場へ一瞬で転移します。
- **`mo-start`**: インタラクティブな起動メニューを表示します。上下キーでお好みのサービス（Frontend, Gateway, AI Engine）を選択し、一括起動できる強力なパッチです。

### 🧠 ② AI エージェント連携と知識同期 (AI Agents & Knowledge Sync)

- **`skill-manager`**: AI エージェントの「武器庫」であるスキルデータベースを管理・更新するための GUI モードを起動します。
- **`sync-chat-docs`**: **【最重要】** プロジェクト内に点在するナレッジドキュメントを 1 つの Markdown ファイルへ自動集約します。これを Claude などの高度な LLM に渡すことで、AI が瞬時にプロジェクトの文脈を理解できるようになります。

### 🧹 ③ 強制クリーンアップ (System Maintenance)

- **`rmf`**: Linux における `rm -rf` に相当する強力な削除ツールです。Windows で頻発する「パスが長すぎて削除できない」`node_modules` などを、問答無用で物理消去します。

---

## 3. 自動化の自動化：AI に設定を任せる

これらの設定ファイルを自分で書き換える必要はありません。

**「AI に指示を出す」** だけで十分です。例えば、Cursor やエージェントに対し、「私の $PROFILE を開き、`mo-api` で gateway ディレクトリに移動する別名を追加して、読み込み直して」と指示してください。AI は正確にパスを特定し、システムに組み込んでくれます。

---

## 💡 アーキテクトによる運用のヒント

優秀なエンジニアは、同じコマンドを 3 回以上打つことがあれば、それを即座に自動化する仕組みを構築します。

ターミナルは、単なる文字の入力画面ではなく、プロジェクトという巨大な母艦を制御するための「統合コックピット」です。自分専用の「黒魔術（ショートカット）」をナレッジベースに蓄積し、AI エージェントと共有し続けることで、あなた自身の開発体験は次のステージへと進化します。
