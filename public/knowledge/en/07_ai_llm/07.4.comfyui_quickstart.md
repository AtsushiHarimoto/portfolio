## 1. Required Course: The "Node Philosophy" of Unpacking the Black Box

Opening ComfyUI for the first time might scare you: the screen is covered in wires, looking like a massive spider web. But once you understand the logic of "Nodes," you'll discover it's the most flexible set of building blocks in the world.

- **Node (Computational Node)**: It's an emotionless worker. One is responsible for loading the model, another for translating your prompt, and another for rendering the raw data into an image.
- **Link (Data Cables)**: These lines between nodes transmit "pure energy and information." When you plug the output of a "Model Node" into the input of a "Sampler Node," sparks fly.
- **Workflow**: This is our "Automation Script." All the connections you make can be saved as a `.json` file. You can load god-tier drawing scripts from anywhere at any time.

## 4. From Coder to Boss: Turning the Engine into Your API Slave

This is where the real value of this lesson lies. How do we make the Moyin backend automatically tell ComfyUI to draw?

### Step 1: Open the Developer Backdoor

Go to Settings and enable **"Enable Dev mode Options."** You will now see a new button called **"Save (API Format)."** The JSON saved here is the true "Programmable Script," stripped of all useless UI coordinates, leaving only the logical skeleton.

### Step 2: Commanding via Python

With just a few lines of code, you can control ComfyUI in the background:

```python
import json, urllib.request

# Load your saved API script
with open("beauty_workflow_api.json") as f:
    wf = json.load(f)

# Change the Prompt to draw a cyber-cat
wf["6"]["inputs"]["text"] = "A cool cyber-cat, neon city, 8k"

# Send it to the ComfyUI server
req = urllib.request.Request("http://127.0.0.1:8188/prompt",
                             data=json.dumps({"prompt": wf}).encode())
urllib.request.urlopen(req)
```

See? That's development! Next time your NPC needs a new outfit, your backend types a few keys, and the AI automatically draws it and sends it back.
