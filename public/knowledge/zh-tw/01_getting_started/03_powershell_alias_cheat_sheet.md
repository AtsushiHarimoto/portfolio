# 03. 終端機裡的黑魔咒：PowerShell 快捷別名指引 (Alias Cheat Sheet)

## @概覽

資深工程師的生產力關鍵不在於打字速度，而在於如何徹底消除重複性的手動操作。在 Moyin 這種涉及多個微服務與複雜目錄鏈的專案中，將冗長的啟動連鎖指令轉化為簡短的「快捷別名 (Alias)」，是進入高效開發狀態的第一步。本篇將引導您建立一套專屬的「終端機控制台」，實現一鍵閃現與瞬時點火。

---

## 1. 拒絕重複勞動：設定 Alias 的必要性

依賴記憶力去背誦深層目錄路徑或複雜的編譯參數是非常低效且容易出錯的。

### 🛠️ 自動化的核心機制

透過修改終端機配置檔（如 Windows PowerShell 的 `$PROFILE` 或 macOS 的 `.zshrc`），我們可以將一長串複雜指令映射至 2~3 個英文字母。

- **優勢**：
  1.  **極致防呆**：將正確的路徑與參數固化為指令，將操作錯誤率降至 0%。
  2.  **肌肉記憶開發**：僅需數秒即可讓完整的開發、AI 演算與測試環境就緒，大幅減少進入工作狀態的摩擦力。

---

## 2. 專案保命符：核心快捷指令清單 (Core Command List)

我們為 Moyin 專案定義了三大戰略模組的快捷指令。強烈建議在環境安裝後立即配置：

### 🚀 ① 瞬間點火與導航 (Startup & Navigation)

- **`mo-api`**：一秒閃現切換至 API 核心目錄 (`projects/moyin-gateway`)，並預備好後端環境。
- **`mo-web`**：一秒切換至前端控制台 (`projects/moyin-web`) 戰場，準備進行 UI 迭代。
- **`mo-start`**：懶人包啟動器。敲下後會彈出互動式選單，供您用上下鍵選擇要啟動哪一個層級的服務。

### 🧠 ② AI 招募與記憶同步 (AI Agents & Knowledge Sync)

- **`skill-manager`**：啟動終端機圖形界面，管理您的 AI 傭兵技能包 (Skills DB)，執行一鍵更新或同步。
- **`sync-chat-docs`**：**【神級指令】**。這支腳本會將專案內分散的知識庫自動組裝並壓縮為單一 Markdown 巨集文字檔。這能讓您在開啟網頁版高階 LLM 時，直接以此檔案作為全量上下文，讓 AI 瞬間通靈掌握專案現狀。

### 🧹 ③ 跨平台系統清理 (System Maintenance)

- **`rmf`**：跨平台強制刪除工具，等同於 Linux 下的 `rm -rf`。專門處理 Windows 中路徑過深或被佔用、無法正常刪除的 `node_modules` 等大型垃圾資料夾。

---

## 3. 把活推給機器：利用 AI 配置環境

您無需精通 PowerShell 語法，只需「出一張嘴」讓 AI 為您服勞役。

您可以直接對您的 AI 助手發出指令：「_請調用終端機存取權，用編輯器打開我的 $PROFILE 檔。幫我加上一個名為 `open-ideas` 的別名，讓我能一鍵切換到 `workspace/ideas` 目錄並啟動 VS Code。_」AI 會精準地完成寫入並重新載入，讓指令即刻生效。

---

## 💡 架構師的コピペ (Copy-paste) 開發觀

優秀的架構師會將重複執行的動作視為一種系統缺陷。

終端機不應只是文字顯示窗口，而應是統御整個專案母艦的「整合駕駛艙」。將這些「黑魔術」指令固化進您的工作流，是從搬磚工人晉升為系統導航員的必經門檻。
