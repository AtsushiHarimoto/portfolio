# 給 AI 裝上最強外掛：Claude Code 技能擴充目錄軍規 (Claude Skills Structure)

## @概覽

大家好，我是 AKIRA。
這期我們來講講，怎麼幫你的 AI 助手（像是 Claude Code 或 Antigravity）掛載那些能毀天滅地的「自定義技能」。

如果你以為幫 AI 擴充能力，就是隨便丟一個寫滿指令的 Word 檔或 Markdown 給它讀，那我只能說：你也太小看這套工程體系了。AI 是很講究「環境感應」的。如果你目錄沒切好、檔案名沒寫對，你就算在對話框裡喉嚨喊破，AI 也絕對抓不到你那招最強大的外掛指令。

今天我們就來聊聊，Moyin 專案對「外部技能包 (Skills)」這套資料夾結構的、如軍規般嚴格的配置標準。

---

## 1. 致命的初學者誤區：單檔集中式陷阱

很多半瓶水的開發者會想：「既然我有五個技能點，那我就在一個檔案裡寫完就好了呀？」他們會建立一個 `.claude/skills/everything.md`，然後在裡面寫滿各種腳本。
**我告訴你，這在現代 AI Agent 掃描機制下是完完全全、徹頭徹尾的錯誤！**

- **正確的鐵律**：為了解鎖那種精確的 `@技能名` 定向觸發能力，**每一個獨立的技能 (Skill)，都必須擁有它自己專屬的實體資料夾目錄！**
- **同步原則**：主開發庫在 `.agent/skills/`，如果你是用 Claude Code 的人，我們會透過腳本一鍵同步到 `.claude/skills/` 下。

---

## 2. 標準資安級目錄結構 (Skills Layout)

請記住這套層級。這是為了能讓 AI 在掃描的一瞬間，就把你的「技能金鑰」存入它的快速緩衝記憶體。

```text
.agent/skills/          # (這是我們項目的技能大本營)
├── handoff/            # [各個技能都有自己的獨立城堡]
│   ├── SKILL.md        # ⬅️ 絕對核心！這是技能的主說明書
│   └── TEMPLATE.md     # 輔助用的各種模板、參考檔案
│
├── new_issue/          # 對應輸入「@new_issue」時的反應
│   └── SKILL.md
│
└── dev/                # 對應輸入「@dev」時的反應
    └── SKILL.md
```

### 為什麼我非得幫每個技能建資料夾不可？

因為高階的技能（比如「跨組交接 `handoff`」）通常不只是一個指令，它還需要帶一些輸出的 JSON 規格或是 Markdown 模板文件。
**只有做成目錄封裝，AI 才能在載入 `@handoff` 的那一刻，把整個資料夾裡的背景資料一次性打包讀取。** 這種封裝性才是企業級開發的底氣。

---

## 3. `SKILL.md`：那張決定成敗的「身分證」

在每個技能目錄下，你必須有一個精確命名的 `SKILL.md`。而這份文件最關鍵的，就是它頭部那個長得像密碼的 **YAML Frontmatter** 區塊。如果你漏了這幾行，AI 掃過去就會把它當成一坨廢紙，理都不理你！

```markdown
---
name: new_issue
description: "協助建立新任務追蹤實體。觸發方式：@new_issue"
---

# 建立全新任務 (new_issue)

（後面才是具體的 SOP 步驟內容...）
```

### 這裡有兩個「沒做對就報銷」的查核點：

1.  **`name`**：必須跟你的資料夾名字一模一樣！這就是 AI 識別你的「呼叫暗號」。
2.  **`description`**：必須要在一句話內講完。而且大家聽好——**末尾一定要明文寫出「觸發方式：@xxx」這幾個字！** 這是一個很重要的語義權重標籤，少了它，AI 懶惰的掃描引擎就可能漏掉這招必殺技。

這才是讓 AI 把你的專案當成它家在打掃的最高心法。當你要求 AI 幫你寫新技能包時，請用出資者的氣場來下令：

> 🗣️ `「AI 助教！你現在要幫我寫一招全新的 @commit 技能外掛！
給我聽清楚這套軍規流程：第一，在 .agent/skills/ 之下建一個名為 commit 的實體目錄！
第二，在裡面給我產出一個 SKILL.md，頭部必須包含對齊目錄名的 YAML Frontmatter，且 description 要寫清楚如何觸發！
第三，不准給我寫那種通篇廢話的教學。我要你直接實作一套能掃描 Git Staging Area 並自動生成 Conventional Commit 訊息的硬核 SOP 教範！
做完後，立刻啟動同步腳本，把這招新技能給我掛載上去生效！去辦！」`

只要目錄跟 Frontmatter 這兩個大門鎖好了，這尊 AI 大腦就能如臂使指地任你差遣。
