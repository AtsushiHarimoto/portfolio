## 1. 入門必修：拆解黑箱的「節點哲學」

很多人第一次打開 ComfyUI 會被嚇跑：螢幕上全是線，跟蜘蛛網一樣。但只要你懂了「節點 (Nodes)」的邏輯，你就會發現它簡直是世界上最自由的積木。

- **Node (運算節點)**：它是個沒感情的工人。一個負責載入模型，一個負責翻譯你的髒話 Prompt，一個負責把雜亂的數據畫成圖片。
- **Link (資料電纜)**：節點之間的那些線，傳輸的是「純粹的能量與資訊」。當你把「模型節點」的輸出插進「渲染節點」的輸入時，火花就產生了。
- **Workflow (工作流)**：這就是我們的「自動化劇本」。所有的連線拓樸最後都會存成一個 `.json` 檔。你可以像換手機鈴聲一樣，隨時載入別處買來的神級畫圖腳本。

---

## 2. 戰鬥準備：打造一個「不搞事」的沙盒環境

在 Windows 上跑這種重型 AI 軟體，最怕的就是把電腦搞髒。聽我的：**一律使用 Portable (免安裝可攜版)！**

```text
ComfyUI_windows_portable/
├── ComfyUI/
│   ├── models/             # 大型 Checkpoint 裝在這裡
│   ├── custom_nodes/       # 各路神人的外掛插件塞這裡
│   └── output/             # 產出的妹子圖片都在這
├── python_embeded/         # 這是它自帶的獨立大腦，絕不干擾你的系統
└── run_nvidia_gpu.bat      # 啟動這台印鈔機的按鈕
```

> 💡 **如果你是顯存較小戶 (< 6GB)**：
> 千萬別硬幹！請在啟動時加上 `--lowvram` 參數。這就像是在引擎上加了個冷卻器，雖然慢一點，但至少不會讓你的顯示卡當場噴煙罷工。

## 3. 必裝神裝：ComfyUI Manager (管理員之眼)

如果你不想在那邊手動下載依賴檔到天荒地老，你進去的第一件事，就是去把 **ComfyUI Manager** 裝起來。

它的功能只有一個：當你從網上抓到一個超酷的工作流 JSON，一打開卻發現滿螢幕都是「紅色報錯節點」時——**不要崩潰！** 點一下 Manager 裡的 **`Install Missing Custom Nodes`**，它就會自己像掃描儀一樣，去全網把那些你缺的插件通通抓回來裝好。這才叫優雅。

---

## 4. 從碼農到老闆：將生成引擎變成你的 API 奴隸

這才是這節課含金量最高的地方。我們怎麼讓 Moyin 的後端自動去叫 ComfyUI 畫圖？

### 第一招：開啟開發者暗門

去 Settings 裡把 **「Enable Dev mode Options」** 打開。這時候你會多出一顆按鈕叫 **「Save (API Format)」**。存下來的 JSON 才是真正的「可程式化劇本」，裡面去掉了所有沒用的 UI 座標，只剩下純粹的邏輯骨架。

### 第二招：用 Python 遠程發號施令

你只要寫幾行腳本，就能在後台「無感」地控制 ComfyUI：

```python
import json, urllib.request

# 載入你剛才存的 API 劇本
with open("beauty_workflow_api.json") as f:
    wf = json.load(f)

# 改掉 Prompt，直接叫它畫個賽博貓咪
wf["6"]["inputs"]["text"] = "A cool cyber-cat, neon city, 8k"

# 甩給 ComfyUI 伺服器
req = urllib.request.Request("http://127.0.0.1:8188/prompt",
                             data=json.dumps({"prompt": wf}).encode())
urllib.request.urlopen(req)
```

看到了嗎？這才叫開發！以後你的遊戲 NPC 想要換套衣服，你後端敲敲鍵盤，AI 就自動畫好傳回來了。
