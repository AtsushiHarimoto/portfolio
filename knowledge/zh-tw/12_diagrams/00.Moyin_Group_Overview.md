## 1. 🥋 幻海師範的道場：Moyin Model Studio

**比喻**: 靈光波動拳修練場
**位置**: `workspace/knowledge/00_projects/moyin-model-studio`
**視覺版圖**: [01.Reikai_Dojo.canvas](../00_projects/moyin-model-studio/01.Reikai_Dojo.canvas)

這裡就像是幻海師父的道場，充滿了嚴酷的試煉。普通的靈氣（數據）在這裡是撐不下去的！
必須經過**基礎靈力訓練 (Pipeline)**，將雜質去除，才能提煉出純正的靈氣。
然後，繼承歷代強者留下的**靈光玉 (Base Models)**，進行最終的**奧義傳承 (Training)**。
只有通過**試煉之門 (Evaluation)** 的強者，才能掌握「靈光波動拳」，成為獨當一面的戰力！

- **Raw Data** -> **靈力資質**
- **Pipeline** -> **基礎訓練**
- **LoRA/Model** -> **靈光波動拳奧義**

## 2. 👁️ 魔界整容師的手術室：Moyin ComfyUI

**比喻**: 時雨的俱樂部
**位置**: `workspace/knowledge/00_projects/moyin-comfyUI`
**視覺版圖**: [01.Makai_Surgery.canvas](../00_projects/moyin-comfyUI/01.Makai_Surgery.canvas)

哎呀，這裡感覺好陰森啊……這裡是魔界整容師時雨的地盤。
參賽者如果想要改變外貌，或者獲得特殊能力（生成圖片），就要來這裡接受改造。
你需要提交強烈的**執念 (Prompt)** 作為代價，時雨會設計精密的**手術術式 (Workflow)**。
有時候還會加上各種**妖力增幅器 (Custom Nodes)**，最後……碰！**具現化 (Generation)** 完成！

- **Prompt** -> **執念/代價**
- **Workflow** -> **手術術式**
- **Image** -> **改造後的肉體**

## 3. 🌹 妖狐藏馬的魔界植物園：Index-TTS

**比喻**: 召喚魔界植物
**位置**: `workspace/knowledge/00_projects/index-tts`
**視覺版圖**: [01.Kurama_Garden.canvas](../00_projects/index-tts/01.Kurama_Garden.canvas)

噓……安靜點。這裡是藏馬的秘密花園。
那些看似美麗的植物，其實都是魔界的生物！
藏馬使用**前世的記憶 (Reference Audio)** 作為觸媒，注入強大的**妖力 (Inference)**。
催生出能夠模仿任何聲音的**魔界含羞草 (Voice Cloning)**。聽到了嗎？那迷人又危險的聲音……

- **Reference Audio** -> **召喚觸媒**
- **Inference** -> **妖力注入**
- **TTS Output** -> **魔界植物的聲音**
