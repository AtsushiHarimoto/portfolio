# 09. Model Studio Integration

## 🧠 模型訓練與數據生成流程

展示 `moyin-model-studio` 如何集成小說流水線與角色音色/形象訓練。

```mermaid
flowchart TD
    Raw[原始小說文本] -->| novel-jsonl-pipeline | Clean[清洗過的 JSONL]

    subgraph "Character Studio"
        Clean -->|抽出角色| Profiles[角色檔案]
        Profiles -->|音色分析| Tone[Tone Annotation]
        Tone -->|模型訓練| GPT_SoVITS[TTS 模型]
    end

    subgraph "Story Studio"
        Clean -->|場景描述| Prompts[SDXL/Wan2.2 Prompts]
        Prompts -->|Lora 訓練| Visual[視覺風格模型]
    end

    GPT_SoVITS & Visual -->|整合| Final[專屬影視模型包]
```

## 🎯 整合目標

1.  **Low-Code Interface**: 使用 VueFlow (Vue 3) 重構訓練流水線。
2.  **Dataset Automation**: 自動將長篇小說轉化為高質量的訓練數據集。
3.  **Cross-Model Alignment**: 確保同一角色的視覺 Lora 與 TTS 音色在生產時高度對齊。
