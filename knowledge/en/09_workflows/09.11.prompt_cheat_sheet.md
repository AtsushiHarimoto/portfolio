# AI Collaboration Command Cheat Sheet

> Forgot a command? Run `./help.ps1` (Win) or `./help.sh` (Mac/Linux) in the project root for an interactive menu.

---

## Core Development

| Task                | Command (Prompt)  | Description                                                      |
| ------------------- | ----------------- | ---------------------------------------------------------------- |
| **Develop feature** | `/run-dev`        | Standard dev SOP: Doc check -> Plan -> Execute -> Verify         |
| **Debug a bug**     | `/run-bg`         | Systematic bug diagnosis: Collect evidence -> Root cause -> Fix  |
| **Run tests**       | `/run-qa`         | Test execution SOP: Create plan -> Run tests -> Generate report  |

## Version Control & Archiving

| Task                | Command (Prompt)       | Description                                                    |
| ------------------- | ---------------------- | -------------------------------------------------------------- |
| **Commit code**     | `/run-archive`         | **Recommended**. Auto-analyze diff, generate `1.repo.md`, git commit |
| **Update log**      | `/run-update-blocks`   | Scan today's commits, auto-update `workspace/work/blocks/`     |

## Knowledge Management & Documentation

| Task                   | Command (Prompt)      | Description                                                    |
| ---------------------- | --------------------- | -------------------------------------------------------------- |
| **Sync docs**          | `/run-sync-docs`      | Full doc backup to `output/` (for GPT Projects)                |
| **Wiki management**    | `/run-wiki`           | Subcommands: `inbox`, `sync`, `conversation`, `index`          |
| **Japanese sync**      | `/deliverables_jap`   | Sync Traditional Chinese docs to Japanese deliverables          |

## Claude Code Specific (VS Code)

Type directly in the Claude conversation box:

| Scope    | Command Example                                              |
| -------- | ------------------------------------------------------------ |
| **GAME** | `GAME - Please add Traditional Chinese comments to src/vn/runtime/...` |
| **WEB**  | `WEB - Please refactor WorldBuilder state management...`     |
| **API**  | `API - Please explain the web2api routing logic...`          |
| **TEST** | `TEST - Please write E2E tests for the new feature...`       |

> Claude Code auto-detects the shorthand and locates the correct working directory.

---

## Tips

1. **Always PLAN first**: Don't let the AI modify code directly. Request `/run-dev` first or manually type `PLAN:`.
2. **Small steps, fast iteration**: One prompt for one task (e.g., modify only one file, or fix only one bug).
3. **Evidence first**: When reporting a bug, use `/run-bg` first and provide logs/screenshots.
