# 04.2. UI スタイルガイド (UI Style Guide)

> **種類**: デザインシステム & テーマ  
> **重点**: Moyin のグローバルビジュアル言語を定義し、Glassmorphism のテクニック、Tailwind のユーティリティ例、3D レイアウトやコピー防止策をまとめます。

---

## 1. 中核となる視覚語彙：Glassmorphism (グラスモーフィズム)

Moyin の美学は現代的で軽やか、かつ多層的な奥行きを帯びています。ビジュアルの核は「フロストガラス」処理で、以下の三層を正確にブレンドします。

1. **半透明のベース**  
   - ダークテーマ：`bg-gray-900/60`  
   - ライトテーマ：`bg-white/40`
2. **光学的ブラー**  
   - `backdrop-blur-md` や `backdrop-blur-xl` を使い、背景をぼかして文字対比を確保します。
3. **エッジハイライト**  
   - `border border-white/10` や `border-blue-100/10` などで薄い縁取りを施し、厚みを感じさせます。

## 2. Tailwind CSS ユーティリティキット

以下のクラス組み合わせを Vue コンポーネントのベースにしてください。

### 2.1 ガラスパネル (Glass Panel)

チャットログ、設定タブ、モーダルなど共通コンテナとして使います。

```html
<div
  class="bg-gray-800/50 backdrop-blur-xl border border-white/10 rounded-xl p-4 shadow-lg"
>
  <!-- コンテンツ -->
</div>
```

### 2.2 グロウ CTA ボタン (Glow Button)

「ゲーム開始」や「送信」などの CTA に、ピンクからインディゴへのグラデーションとホバーグロウを付与します。

```html
<button
  class="bg-gradient-to-r from-pink-500 to-blue-500 text-white font-bold py-2 px-4 rounded-lg shadow-lg hover:brightness-110 transition-all duration-300"
>
  実行
</button>
```

### 2.3 フォーカス入力欄 (Active Input Field)

背景は濃く、フォーカス時にピンクリングをまとうことで明確なフィードバックを提供します。

```html
<input
  class="bg-black/20 border border-white/10 rounded-lg px-4 py-2 text-white focus:border-pink-500 focus:ring-1 focus:ring-pink-500 outline-none transition-all"
  placeholder="対話を入力..."
/><!-- /-->
```

## 3. レイアウトのベストプラクティス

- **本文のコントラスト調整** – ダークモードで白文字 (`text-white`) は長時間でまぶしすぎるため、`text-gray-200` か `text-gray-300` に落とします。  
- **余白とネガティブスペース** – `gap-4`, `p-6`, `my-4` などを守り、要素同士が詰まりすぎないようにします。  
- **レスポンシブグリッド** – `grid-cols-1 md:grid-cols-2` をデフォルトとし、モバイル時に自然に折りたたむよう構成します。

---

## 4. Light Breeze バリアント

`moyin-novel` のような、テキストを主体とした没入型ブランチ向けの視覚セットです。

- **空色グラデーション**：重い黒を捨てて `linear-gradient(135deg, #f0f9ff 0%, #bae6fd 100%)` を採用。  
- **ジェリーパネル**：不透明度を `rgba(255, 255, 255, 0.5)` に上げ、角丸を `12px` に。  
- **朝露の影**：濃い影の代わりに `rgba(0, 50, 100, 0.05)` の淡いブルー影で、画面を清新に保ちます。

---

## 5. WebGL / Three.js の共存

- **Glow の連動**：Three.js の `ShaderMaterial` でフレネル法線を使い、UI のグロウと調和させます。  
- **Z-index 分離**：3D キャンバスは `z-index: -1` に固定し `pointer-events: none` を付与して Vue UI のイベントを邪魔させません。

---

## 6. 著作権防御

1. **選択 & 右クリック禁止**：`-webkit-user-select: none;` と `contextmenu` 監視でコピーを封じます。  
2. **開発者ツールの遮断**：`F12`, `Ctrl+Shift+I`, `Ctrl+U` を検知して無効化します。  
3. **動的ウォーターマーク**：Canvas でユーザーIDを含む斜め文字を描き、`opacity < 0.03` で配置。目にはほぼ見えませんが、漏洩時には追跡可能です。
